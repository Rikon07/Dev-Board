There are several ways to select an element in DOM.
  • getElementById()
  • getElementsByClassName()
  • getElementsByTagName()
  • querySelector()
  • querySelectorAll()
  • closest()
The differences are:
  • innerHTML sets HTML content, including tags.
  • innerText sets only the visible text, considering CSS styles.
  • textContent is raw text, unaffected by CSS styles and includes hidden text.
Event Delegation is a JavaScript feature where instead of adding an event listener to each child element, event listener can be added only to the parent element. This technique helps to manage the events of child elements effectively, specially for dynamic uses and deleting or adding to the DOM.
Event bubbling refers to an event which is triggered on an element that will bubble up to its parent elements. It helps handling events effectively, especially in cases where there are lot of elements or dynamic contents.
Using JavaScript =>
  • Elements are created using document.createElement()
  • Elements are added using append() or appendChild()
  • Elements are removed using removeChild()
